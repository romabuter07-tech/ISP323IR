# Введение 
Проект «Калькулятор индекса массы тела» представляет собой веб-приложение, предназначенное для расчёта показателя BMI (Body Mass Index).

Индекс массы тела используется для оценки соответствия веса человека его росту и позволяет определить категорию веса: недостаточная масса тела, нормальный вес, избыточная масса или ожирение. 

Формула расчёта: **BMI = вес (кг) / (рост (м) × рост (м))** 

Приложение обеспечивает быстрый и удобный способ получения результата без выполнения вычислений вручную.

# Примеры использования 

1. При вводе роста в сантиметрах система принимает значение для расчёта.   

2. При вводе веса в килограммах данные сохраняются в поле ввода.   

3. При нажатии на кнопку «Рассчитать» выполняется вычисление индекса массы тела.   

4. При корректном вводе данных отображается результат с точностью до двух знаков после запятой.   

5. При значении BMI менее 18.5 выводится сообщение «Недостаточная масса тела».   

6. При значении BMI от 18.5 до 24.9 отображается сообщение «Нормальный вес».   

7. При значении BMI от 25 до 29.9 выводится сообщение «Избыточная масса тела».   

8. При значении BMI 30 и выше отображается сообщение «Ожирение».   

9. При вводе отрицательных значений программа выводит сообщение об ошибке.   

10. При вводе текстовых данных вместо числовых система предупреждает о некорректном формате.   

11. При нажатии на кнопку «Очистить» все поля ввода автоматически обнуляются.   

12. При обновлении страницы введённые данные сбрасываются.
